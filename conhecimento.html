Linguagem C

A Linguagem C é a que eu mais utilizei até o momento pois ela está sendo o foco para aprendizado na minha graduação. Eu fiz muitos exercícios sobre vetores, laços de repetição, condições If/Else, funções, ponteiros, strings, modos de ordenação, pilhas, listas lineares, arquivos .txt/.bin, entre outros conceitos.

Linguagem Python

Embora não tenha muita experiência com a linguagem, fiz o curso do canal "Curso em Video" no youtube e tento aprender um pouco mais nas minhas horas vagas. Infelizmente não há muitas vagas de trabalho para essa linguagem no Brasil, o que me levou a deixa-la um pouco de lado pois estou priorizando linguagens mais populares, visando entrar no mercado de trabalho. Mesmo assim, pratiquei alguns conceitos Pythonicos como: condições If/Else, dicionários, laços de repetição, strings, tuplas, strings, módulos, etc. Também fiz um projeto utilizando o conceito Web Crawler, onde o programa acessa o site de uma empresa e retorna com a informação do dólar atualizado (Clique aqui para ter acesso ao código no Github). Realmente é uma ótima linguagem para buscar dados em páginas na internet e espero um dia ter a oportunidade de me aprofundar nela. Clique aqui para acessar o meu repósitório sobre Python no Github.
